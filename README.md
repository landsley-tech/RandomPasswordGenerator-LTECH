# RandomPasswordGenerator-LTECH

This password generator was created to help me learn C. 

Features:
- Selective number of characters
- Completely Random
- Seed for randomness based on time of day
- Password generation includes typical password requirements


HOW TO USE

Copy code into a compiler
Run the code
Enter total characters for password
It will generate a password
